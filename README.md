## 写作规范

本书遵循 [Markdown 写作组件规范](../AI神经网络的数学-用Python、PyTorch讲解-Markdown写作规范.md)：

- **数学公式**：行内 `$...$`，行间 `$$...$$`
- **代码块**：标注语言（python / output / bash）
- **提示块**：`> **核心洞察**`、`> **提示**`、`> **警告**`、`> **注意**`
- **小精灵**：`> **小精灵说**`
- **插图**：`![图 n-m：描述](相对路径)`

## 核心理念

1. **数学直觉比公式推导更重要**——每个公式都有「人话解释」
2. **代码是用来验证直觉的，不是主角**——先直觉，后公式，再代码
3. **理解 = 知道「为什么」> 知道「是什么」**——每个子节回答「为什么需要这个」
